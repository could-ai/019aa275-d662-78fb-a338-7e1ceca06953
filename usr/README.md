# Ali Poster Creator

تطبيق لتصميم بوسترات جولات تيك توك (TikTok) الرسمية.

## المميزات
- 15 قالب لوني جاهز (تدرجات ألوان تيك توك).
- إمكانية إضافة صور المتنافسين.
- تعديل الأسماء، التوقيت، والدولة.
- تاريخ تلقائي.
- شعار "جولة رسمية" وتوقيع "Ali Poster Creator".

## كيفية التشغيل (How to Run)

1. **تثبيت المكتبات (Install Dependencies):**
   افتح التيرمينال (Terminal) واكتب الأمر التالي لتثبيت المكتبات المطلوبة:
   ```bash
   flutter pub get
   ```

2. **تشغيل التطبيق (Run App):**
   لتشغيل التطبيق على المتصفح (Chrome) أو المحاكي، اكتب:
   ```bash
   flutter run
   ```

## ملاحظات
- لتشغيل التطبيق على الويب بشكل أفضل، استخدم:
  ```bash
  flutter run -d chrome
  ```
